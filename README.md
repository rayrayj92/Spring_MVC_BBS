# Spring_MVC_BBS

## 개요
스프링 프레임워크를 사용하여 게시판 기능을 구현해보았습니다. [자바 어노테이션 기반 설정]

## 기술
* Spring Framework 5.2.5
* MySQL 8.0.19
* Hibernate 5.4.12
* Bootstrap 4.4.1
* CK Editor 4.14.0

## 기능
* 게시판
    * Input 값 유효성 체크[Back-End, Front-End(HTML5 Attribute)]
    * 검색(Search)
        * 제목 및 작성자로 검색
    * 작성(Create)
        * 위지위그(CK Editor)
    * 조회(Read)
        * 조회수 + 1 Update
    * 수정(Update)
        * 첨부파일 수정 및 저장
        * 위지위그(CK Editor)
    * 삭제(Delete)
        * 일괄삭제[Batch Delete]
